Project Title: Wine Quality Prediction

Description: This project focuses on building a classification model using Random Forest Classifier to predict the quality of wine based on various physicochemical properties. It includes data preprocessing, exploratory data analysis (EDA), model training, and a predictive system for real-time quality prediction.


Workflow Summary: Data Exploration, Data Visualization (EDA), Data Preparation, Model Building, Prediction System.

Tools & Libraries Used: Python, 

NumPy

,Pandas, 

Seaborn, 

Matplotlib
, 
Scikit-learn (RandomForestClassifier, train_test_split, accuracy_score).
